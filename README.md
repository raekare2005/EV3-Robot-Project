# EV3-Robot-Project
# 🤖 乐高 EV3 智能机器人综合控制系统

> 本仓库包含了基于乐高 MINDSTORMS EV3 (EV3 Classroom) 图形化编程平台开发的智能机器人控制程序 (`.lmsp` 格式)。项目集成了自动巡线、颜色识别、动态避障以及高精度姿态控制等多个高级功能模块。

## 💡 项目核心功能 (Modules)

本项目包含 4 个独立的子程序模块，分别对应不同的复杂任务场景：

### 🔴 模块一：颜色识别与动态统计 (Module 1 - Color Detection)
* **功能描述**：机器人在行进过程中实时扫描地面颜色。
* **技术实现**：利用颜色传感器检测红、黄、蓝、绿四种颜色的出现次数，并在 EV3 主机屏幕上分行实时滚动显示统计数据。当传感器识别到特定颜色边界（如白色/灰色）时，系统会自动执行减速或停止指令。并配有不同颜色的专属提示音效。

### 〰️ 模块二：PID 巡线与自主导航 (Module 2 - PID Auto-Navigation)
* **功能描述**：本项目中最核心的控制模块，使机器人能够极度平滑地沿着复杂轨迹行驶。
* **技术实现**：
  * **PID 控制算法**：摒弃了简单的“非黑即白”巡线，通过读取颜色传感器的“反射光强度”，自主编写了 `SETPID` 和 `PID` 函数。利用比例(P)、积分(I)、微分(D)共同调节左右电机的输出速度，大幅减少了机器人的行驶震荡，实现了丝滑巡线。
  * **节点导航**：结合陀螺仪和颜色传感器，机器人在遇到特定颜色标记（如红区、黑区）时，能自动脱离巡线模式，执行精确的固定角度转向，随后重新切回 PID 巡线模式，完成复杂的迷宫/赛道寻路。

### 🛡️ 模块三：安全距离动态保持 (Module 3 - Maintaining Safe Distance)
* **功能描述**：实现机器人的“自适应巡航”与防碰撞保护。
* **技术实现**：利用超声波传感器实时测距。当与前方障碍物距离小于危险阈值（10.5 cm）时，机器人会自动刹车甚至倒车退让（维持在 10-11 cm 的缓冲警告区），防止由于惯性导致的碰撞。同时，程序在后台全程记录并会在屏幕上输出本次运行的“最大安全距离”与“最小危险距离”。

### 🔄 模块四：直线定姿与 360° 补偿转向 (Module 4 - Precision Steering)
* **功能描述**：高精度的定点姿态控制。
* **技术实现**：结合陀螺仪传感器，机器人直线行驶至黑色标记线后，执行原地 360° 旋转。**亮点在于**：考虑到实际地面摩擦力的物理干扰，程序中对转向角度进行了精准的工程标定（补偿设定为 341°），确保机器人能在物理世界中完美完成 360° 掉头，随后继续向红色终点线冲刺。

## ⚙️ 硬件端口配置建议 (Hardware Setup)

为确保 `.lmsp` 程序正常运行，请按以下默认配置连接您的 EV3 硬件：
* **驱动电机 (Motors)**：端口 B (左电机)、端口 C (右电机)
* **陀螺仪传感器 (Gyro Sensor)**：端口 2
* **颜色传感器 (Color Sensor)**：端口 3
* **超声波传感器 (Ultrasonic Sensor)**：端口 4

## 🚀 如何使用
1. 克隆或下载本仓库的文件。
2. 确保您的设备上已安装 [LEGO® MINDSTORMS® Education EV3 Classroom](https://education.lego.com/) 软件。
3. 双击打开对应的 `1.lmsp` ~ `4.lmsp` 文件即可查看完整代码块并将其下载至 EV3 主机。
